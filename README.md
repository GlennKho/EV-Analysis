APLC Assignment Phase 1: Working with a JSON file containing 3022 records of electronic vehicles (EVs) and plug-in models across many brands over the years, with data on technical specifications, performance, price, origin, yearly sales, and safety-related attributes.
## Questions
Using imperative programming approach, do the following operations:
1. Display the total number of vehicles produced by user-selected manufacturing company.
2. Search and display the list of models available from a user-selected manufacturing company.
3. Identify the model with the longest driving range from user-selected manufacturer.
4. Determine average charging time by charging type
5. Rank 2025 vehicles by safety rating (top 5 safest)
6. Identify the best-selling electronic vehicles in 2024
## How to run
1. Ensure that node is installed on the system
2. Download the electric_vehicles_dataset.json and put into the root folder
3. Using terminal, navigate to the root folder and run 'npm i' to install all dependencies.
4. Start the server by running 'npm run start' or 'node server.js'
5. If successful, Server should start in port 1408. Access the page by going to http://localhost:1408
